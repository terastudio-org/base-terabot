<h1 align="center">🤖 Base ResBot</h1>

<p align="center">
  <b>Library WhatsApp Bot berbasis Baileys — dibuat untuk kecepatan, stabilitas, dan fleksibilitas.</b>
</p>

</p>

---

## 🚀 Fitur Utama

- ⚡ **Koneksi Cepat & Stabil** — Menggunakan Baileys Multi-Device API terbaru
- 🧠 **Serializer Canggih** — Mempermudah akses pesan dan metadata
- 🧩 **Modular System** — Struktur kode mudah dikembangkan
- 🪄 **Media Utils** — Download dan manipulasi media (gambar, video, audio)
- 🔐 **Cache System** — Pengelolaan data grup, user, dan session otomatis
